# Super-Store-Sales-Dashboard
This project presents an interactive Super Store Sales Dashboard developed using Power BI to analyze sales performance, profit trends, and customer behavior in a clear and visually engaging way. The main objective of this dashboard is to transform raw sales data into meaningful insights. that can help businesses understand their overall performance
This Power BI dashboard analyzes Super Store sales data and provides key insights through interactive visualizations.
